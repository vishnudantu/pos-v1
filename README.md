# NETHRA POS-v1 | Political Operating System

Real-time Political Intelligence and Command Platform

## Vision

A JARVIS-like operating system for politicians providing:
- Real-time constituency intelligence
- AI-powered insights and alerts
- Ground-level booth management
- Election war room capabilities

## Architecture

- apps/web: React + Vite + Tailwind
- apps/api: Node.js + Express
- apps/worker: Background jobs
- packages: Shared libraries
- infrastructure: DevOps configs

## Quick Start

1. npm install
2. npm run dev
3. npm run build
